# transportadores-envia-prestashop-9

Modulo para la generación y calculo de envíos de prestashop 9.

## Módulo Envia Shipping

El módulo `envia_shipping` integra la API de [Envia.com](https://envia.com) con PrestaShop 9 para calcular tarifas de envío en tiempo real durante el checkout.

Consulta la documentación completa en [`modules/envia_shipping/README.md`](modules/envia_shipping/README.md).
